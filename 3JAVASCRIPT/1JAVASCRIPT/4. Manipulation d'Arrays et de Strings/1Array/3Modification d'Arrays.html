1. Méthode push()

La méthode push() permet d'ajouter un ou plusieurs éléments à la fin d'un tableau. Voici un exemple simple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
fruits.push('fraise', 'raisin');
console.log(fruits);
// Résultat : ['pomme', 'orange', 'banane', 'fraise', 'raisin']

2. Méthode unshift()

La méthode unshift() ajoute un ou plusieurs éléments au début du tableau. Voici un exemple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
fruits.unshift('fraise', 'raisin');
console.log(fruits);
// Résultat : ['fraise', 'raisin', 'pomme', 'orange', 'banane']

3. Spread Operator (...)

L'opérateur de propagation (...) permet d'étendre un tableau en incluant ses éléments dans un nouveau tableau. Il peut être utilisé pour ajouter des éléments à un tableau existant :

javascript
let fruits = ['pomme', 'orange', 'banane'];
let nouveauxFruits = ['fraise', 'raisin', ...fruits];
console.log(nouveauxFruits);
// Résultat : ['fraise', 'raisin', 'pomme', 'orange', 'banane']

4. Méthode concat()

La méthode concat() crée un nouveau tableau en combinant les tableaux existants. Elle peut être utilisée pour ajouter des éléments à la fin d'un tableau :

javascript
let fruits = ['pomme', 'orange', 'banane'];
let nouveauxFruits = fruits.concat('fraise', 'raisin');
console.log(nouveauxFruits);
// Résultat : ['pomme', 'orange', 'banane', 'fraise', 'raisin']

Ces différentes méthodes offrent des approches variées pour ajouter des éléments à un tableau en JavaScript. Vous pouvez choisir celle qui convient le mieux à votre situation spécifique.

1. Modification directe par index

La modification directe par index est la méthode la plus simple et directe pour changer la valeur d'un élément dans un tableau. Voici un exemple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
fruits[1] = 'fraise';
console.log(fruits); // Résultat : ['pomme', 'fraise', 'banane']

2. Méthode map()

La méthode map() crée un nouveau tableau en appliquant une fonction à chaque élément du tableau d'origine. Elle est souvent utilisée pour la modification d'éléments. Voici un exemple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
let nouveauxFruits = fruits.map(fruit => {
  if (fruit === 'orange') {
    return 'citron';
  } else {
    return fruit;
  }
});
console.log(nouveauxFruits); // Résultat : ['pomme', 'citron', 'banane']

3. Utilisation de forEach()

La méthode forEach() itère sur chaque élément du tableau et permet d'effectuer des modifications directes. Voici un exemple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
fruits.forEach((fruit, index, arr) => {
  if (fruit === 'orange') {
    arr[index] = 'citron';
  }
});
console.log(fruits); // Résultat : ['pomme', 'citron', 'banane']

4. Utilisation de indexOf() et splice()

Vous pouvez utiliser indexOf() pour trouver l'index d'un élément dans le tableau, puis utiliser splice() pour modifier l'élément. Voici un exemple :

javascript
let fruits = ['pomme', 'orange', 'banane'];
let indexOrange = fruits.indexOf('orange');
if (indexOrange !== -1) {
  fruits.splice(indexOrange, 1, 'citron');
}
console.log(fruits); // Résultat : ['pomme', 'citron', 'banane']

Ces méthodes offrent différentes approches pour la modification d'éléments dans un tableau en JavaScript. Choisissez celle qui correspond le mieux à votre besoin spécifique.

1. Shallow Copy vs Deep Copy

Shallow Copy : Une copie superficielle ne crée qu'une nouvelle référence au tableau d'origine. Les tableaux imbriqués (tableaux à l'intérieur d'un tableau) ne sont pas clonés de manière approfondie, ce qui signifie que les modifications apportées à ces tableaux imbriqués se reflètent dans la copie et vice versa.

Deep Copy : Une copie en profondeur crée une nouvelle copie complète du tableau, y compris les tableaux imbriqués. Ainsi, les modifications apportées à un tableau ou à ses éléments ne se reflètent pas dans la copie et vice versa.

2. Méthode slice()

La méthode slice() crée une nouvelle copie d'une partie d'un tableau. Si elle est utilisée sans arguments, elle copie l'ensemble du tableau. Voici un exemple :

javascript
let originalArray = ['a', 'b', 'c'];
let shallowCopy = originalArray.slice();
console.log(shallowCopy); // Résultat : ['a', 'b', 'c']

3. Spread Operator (...)

L'opérateur de propagation (...) peut également être utilisé pour créer une copie superficielle ou profonde d'un tableau. Voici un exemple de copie superficielle :

javascript
let originalArray = ['a', 'b', 'c'];
let shallowCopy = [...originalArray];
console.log(shallowCopy); // Résultat : ['a', 'b', 'c']

Pour une copie en profondeur, assurez-vous que le tableau original n'a pas de références partagées :

javascript
let originalArray = [['a'], ['b'], ['c']];
let deepCopy = JSON.parse(JSON.stringify(originalArray));
console.log(deepCopy); // Résultat : [['a'], ['b'], ['c']]

Note : La méthode JSON.stringify suivie de JSON.parse est une solution simple pour effectuer une copie en profondeur, mais elle a quelques limitations et ne fonctionne pas bien avec certains types de données spécifiques.

4. Utilisation de concat()

La méthode concat() crée une nouvelle copie en combinant plusieurs tableaux. Elle peut être utilisée pour créer une copie superficielle :

javascript
let originalArray = ['a', 'b', 'c'];
let shallowCopy = [].concat(originalArray);
console.log(shallowCopy); // Résultat : ['a', 'b', 'c']

Ces différentes méthodes offrent des approches variées pour copier des tableaux en JavaScript. Choisissez celle qui convient le mieux à votre cas d'utilisation spécifique. En général, pour les copies en profondeur, en particulier avec des structures de données complexes, des bibliothèques comme Lodash peuvent être utiles.

La méthode map() crée un nouveau tableau en appliquant une fonction à chaque élément du tableau d'origine. Cela permet de transformer chaque élément du tableau selon la logique définie dans la fonction. Voici un exemple :

javascript
let nombres = [1, 2, 3, 4, 5];

2. Méthode filter()

La méthode filter() crée un nouveau tableau contenant uniquement les éléments qui passent un test spécifique fourni par une fonction. Cela permet de filtrer les éléments du tableau selon un critère défini. Voici un exemple :

javascript
let nombres = [1, 2, 3, 4, 5];
let nombresPairs = nombres.filter(nombre => nombre % 2 === 0);
console.log(nombresPairs); // Résultat : [2, 4]

3. Méthode reduce()

La méthode reduce() permet de réduire un tableau à une seule valeur en appliquant une fonction cumulative à chaque élément du tableau. Cela peut être utilisé pour effectuer des opérations telles que la somme, la multiplication, la concaténation, etc. Voici un exemple :

javascript
let nombres = [1, 2, 3, 4, 5];
let somme = nombres.reduce((acc, nombre) => acc + nombre, 0);
console.log(somme); // Résultat : 15

Dans cet exemple, la fonction cumulative prend deux paramètres : l'accumulateur (acc) et l'élément actuel (nombre). L'accumulateur conserve le résultat intermédiaire de la réduction, et la valeur initiale de l'accumulateur est spécifiée comme deuxième argument de reduce() (dans cet exemple, 0).

La méthode sort() est utilisée pour trier les éléments d'un tableau. Cependant, cela peut ne pas donner le résultat attendu pour les nombres. Voici un exemple :

javascript
let nombres = [10, 2, 5, 1, 9];

2. Utilisation de compare function

Pour trier correctement les éléments, vous pouvez utiliser une fonction de comparaison personnalisée avec sort(). La fonction de comparaison doit renvoyer un nombre négatif si a < b, zéro si a équivaut à b, et un nombre positif si a > b. Voici un exemple pour trier correctement les nombres :

javascript
let nombres = [10, 2, 5, 1, 9];
nombres.sort((a, b) => a - b);
console.log(nombres); // Résultat : [1, 2, 5, 9, 10]

Si vous voulez trier dans l'ordre décroissant, vous pouvez simplement changer l'ordre des éléments dans la fonction de comparaison :

javascript
let nombres = [10, 2, 5, 1, 9];
nombres.sort((a, b) => b - a);
console.log(nombres); // Résultat : [10, 9, 5, 2, 1]

javascript
let fruits = ['orange', 'pomme', 'banane', 'fraise'];
fruits.sort((a, b) => a.localeCompare(b));

let nombres = [1, 2, 3];

1. Éviter les modifications directes d'Arrays

Modifier directement un tableau peut conduire à des effets de bord inattendus, en particulier lorsque plusieurs parties du code manipulent le même tableau. Préférez plutôt créer de nouveaux tableaux pour éviter des comportements indésirables. Par exemple, évitez de faire :

javascript
let fruits = ['pomme', 'orange', 'banane'];
fruits.push('fraise'); // Modification directe
console.log(fruits); // ['pomme', 'orange', 'banane', 'fraise']

Préférez plutôt :

javascript
let fruits = ['pomme', 'orange', 'banane'];
let nouveauxFruits = [...fruits, 'fraise']; // Création d'un nouveau tableau
console.log(fruits); // ['pomme', 'orange', 'banane']
console.log(nouveauxFruits); // ['pomme', 'orange', 'banane', 'fraise']

2. Utilisation de fonctions immuables

Les fonctions immuables, qui ne modifient pas directement les données mais créent plutôt de nouvelles données, contribuent à écrire un code plus prévisible et facile à comprendre. Les méthodes comme map(), filter(), et reduce() sont utiles pour créer de nouvelles versions des données sans les modifier directement.

Exemple d'utilisation de map() :

javascript
let nombres = [1, 2, 3];
let doubles = nombres.map(nombre => nombre * 2);
console.log(doubles); // [2, 4, 6]
console.log(nombres); // [1, 2, 3]

3. Gestion des valeurs null et undefined

Il est souvent judicieux de vérifier si une valeur est null ou undefined avant de l'utiliser, pour éviter des erreurs inattendues. Vous pouvez utiliser des opérateurs logiques, tels que l'opérateur de coalescence nulle (??), pour fournir une valeur de remplacement si la valeur est null ou undefined.

Exemple :

javascript
let valeurPossiblementNulle = null;
let valeurDeSecours = 'Valeur par défaut';
let resultat = valeurPossiblementNulle ?? valeurDeSecours;
console.log(resultat); // 'Valeur par défaut'

Ces bonnes pratiques et astuces contribuent à rendre votre code JavaScript plus robuste, lisible et facile à entretenir. Adoptez-les dans vos projets pour améliorer la qualité de votre code.
