1. Introduction aux Objets en JavaScript

1.1 Définition des objets

En JavaScript, les objets sont des structures de données complexes qui regroupent des valeurs et des fonctionnalités connexes. Ils sont constitués de paires clé-valeur, où la clé est une chaîne de caractères qui identifie la propriété, et la valeur peut être de n'importe quel type de données.

javascript
// Exemple d'objet
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022,
    démarrer: function() {
        console.log("Le moteur démarre!");
    }
};

1.2 Création d'objets

Il existe plusieurs façons de créer des objets en JavaScript. L'une d'entre elles est l'utilisation de la notation littérale d'objet.

javascript
// Notation littérale d'objet
let personne = {
    nom: "John",
    age: 30,
    adresse: {
        rue: "123 Main St",
        ville: "Villeville",
        codePostal: "12345"
    }
};

Une autre façon de créer des objets est d'utiliser le constructeur Object.

javascript
// Utilisation du constructeur Object
let livre = new Object();
livre.titre = "JavaScript 101";
livre.auteur = "Jane Doe";
livre.pages = 200;

1.3 Propriétés et méthodes

Les objets peuvent avoir des propriétés qui stockent des valeurs et des méthodes qui sont des fonctions liées à l'objet.

javascript
// Propriétés et méthodes d'un objet
let ordinateur = {
    marque: "Dell",
    modèle: "XPS",
    allumer: function() {
        console.log("L'ordinateur est allumé!");
    },
    éteindre: function() {
        console.log("L'ordinateur est éteint.");
    }
};

// Accéder aux propriétés
console.log(ordinateur.marque); // Affiche "Dell"

// Appeler une méthode
ordinateur.allumer(); // Affiche "L'ordinateur est allumé!"

2. Propriétés d'Objet en JavaScript

2.1 Accès aux propriétés

Pour accéder aux propriétés d'un objet, vous pouvez utiliser la notation pointée ou la notation entre crochets.

javascript
// Notation pointée
let personne = {
    nom: "John",
    age: 30,
    adresse: {
        rue: "123 Main St",
        ville: "Villeville"
    }
};

console.log(personne.nom); // Affiche "John"
console.log(personne.adresse.ville); // Affiche "Villeville"

// Notation entre crochets
let propriete = 'nom';
console.log(personne[propriete]); // Affiche "John"

2.2 Modification des propriétés

Pour modifier la valeur d'une propriété, utilisez la notation pointée ou entre crochets.

javascript
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022
};

// Modification avec la notation pointée
voiture.modèle = "Corolla";

// Modification avec la notation entre crochets
voiture['année'] = 2023;

2.3 Ajout et suppression de propriétés

Vous pouvez ajouter de nouvelles propriétés à un objet en utilisant la notation pointée ou en utilisant la notation entre crochets.

javascript
let personne = {
    nom: "John",
    age: 30
};

// Ajout d'une nouvelle propriété
personne.profession = "Développeur";
// ou
personne['adresse'] = "123 Main St";

Pour supprimer une propriété, utilisez l'opérateur delete.

javascript
delete personne.age; // Supprime la propriété 'age'

2.4 Propriétés calculées

Les propriétés calculées vous permettent de définir dynamiquement le nom de la propriété.

javascript
let typePropriete = "couleur";

let maison = {
    rue: "456 Oak St",
    [typePropriete]: "rouge"
};

console.log(maison.couleur); // Affiche "rouge"

3. Méthodes d'Objet en JavaScript

3.1 Méthodes intégrées

JavaScript propose de nombreuses méthodes intégrées que vous pouvez utiliser sur des objets. Par exemple, les méthodes de chaîne de caractères, les méthodes de tableau, etc.

javascript
let message = "Hello, World!";

// Utilisation d'une méthode de chaîne de caractères
let longueurMessage = message.length; // Obtient la longueur du message

Vous pouvez définir des méthodes pour un objet en incluant des fonctions en tant que valeurs de propriétés.

javascript
let personne = {
    nom: "John",
    age: 30,
    afficherDetails: function() {
        console.log(`Nom: ${this.nom}, Âge: ${this.age}`);
    }
};

// Appel de la méthode définie
personne.afficherDetails(); // Affiche "Nom: John, Âge: 30"

3.3 "this" dans les méthodes

Dans une méthode, le mot-clé this fait référence à l'objet sur lequel la méthode est appelée. Il vous permet d'accéder aux autres propriétés de l'objet.

javascript
let compteBancaire = {
    solde: 1000,
    déposer: function(montant) {
        this.solde += montant;
        console.log(`Nouveau solde: ${this.solde}`);
    }
};

compteBancaire.déposer(500); // Augmente le solde de 500

Vous pouvez appeler des méthodes d'objet de différentes manières, y compris en utilisant la notation pointée ou la notation entre crochets.

javascript
let personne = {
    nom: "John",
    direSalut: function() {
        console.log(`Salut, je m'appelle ${this.nom}`);
    }
};

personne.direSalut(); // Affiche "Salut, je m'appelle John"
// ou
personne['direSalut'](); // Affiche "Salut, je m'appelle John"

4. Parcourir les Objets en JavaScript

4.1 Boucle "for...in"

La boucle for...in est utilisée pour parcourir les propriétés d'un objet. Elle itère sur toutes les propriétés énumérables de l'objet.

javascript
let voiture = {
    marque: "Toyota",
    modèle: "Camry",

// Affiche:
// marque Toyota
// modèle Camry

4.2 Object.keys(), Object.values(), Object.entries()

Les méthodes Object.keys(), Object.values(), et Object.entries() peuvent être utilisées pour obtenir des tableaux de clés, de valeurs, ou de paires clé-valeur d'un objet respectivement.

javascript
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022
};

// Obtient un tableau de clés
let cles = Object.keys(voiture);
console.log(cles); // Affiche ["marque", "modèle", "année"]

// Obtient un tableau de valeurs
let valeurs = Object.values(voiture);

// Obtient un tableau de paires clé-valeur
let paires = Object.entries(voiture);

4.3 La boucle "for...of" pour les valeurs

La boucle for...of peut être utilisée pour itérer sur les valeurs d'un tableau ou d'un objet itérable.

javascript
let fruits = ['pomme', 'orange', 'banane'];

for (let fruit of fruits) {
    console.log(fruit);
}
// Affiche:
// pomme
// orange
// banane

Cependant, la boucle for...of ne peut pas être utilisée directement pour parcourir les propriétés d'un objet. Vous devez utiliser Object.values() ou Object.entries() avec la boucle for...of pour obtenir les valeurs.

javascript
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022
};

for (let valeur of Object.values(voiture)) {
    console.log(valeur);
}
// Affiche:
// Toyota
// Camry
// 2022

5. Objets et JSON en JavaScript

5.1 Sérialisation et désérialisation JSON

La sérialisation JSON consiste à convertir un objet JavaScript en une chaîne JSON, tandis que la désérialisation consiste à convertir une chaîne JSON en un objet JavaScript.

javascript
// Sérialisation JSON
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022
};

let voitureJSON = JSON.stringify(voiture);
console.log(voitureJSON);
// Affiche '{"marque":"Toyota","modèle":"Camry","année":2022}'

// Désérialisation JSON
let objetVoiture = JSON.parse(voitureJSON);
console.log(objetVoiture);
// Affiche { marque: 'Toyota', modèle: 'Camry', année: 2022 }

5.2 Méthodes JSON

L'objet JSON en JavaScript fournit des méthodes pour travailler avec JSON.

  • JSON.stringify(objet[, replacer[, espace]]): Convertit un objet en une chaîne JSON.
    • replacer: Une fonction qui peut être utilisée pour transformer les valeurs ou les propriétés lors de la sérialisation.
    • espace: Un argument optionnel pour définir l'espacement dans la chaîne JSON résultante.
javascript
let voiture = {
    marque: "Toyota",
    modèle: "Camry",
    année: 2022
};

let voitureJSON = JSON.stringify(voiture, null, 2);
console.log(voitureJSON);
// Affiche la chaîne JSON avec un espacement de 2 caractères
  • JSON.parse(chaine[, reviver]): Convertit une chaîne JSON en un objet JavaScript.
    • reviver: Une fonction qui peut être utilisée pour transformer les valeurs pendant la désérialisation.
javascript
let voitureJSON = '{"marque":"Toyota","modèle":"Camry","année":2022}';
let objetVoiture = JSON.parse(voitureJSON);
console.log(objetVoiture);
// Affiche { marque: 'Toyota', modèle: 'Camry', année: 2022 }

6. Objets Avancés en JavaScript

6.1 Héritage

JavaScript prend en charge l'héritage à l'aide de prototypes. Vous pouvez créer des objets qui héritent des propriétés et des méthodes d'un autre objet.

javascript
// Fonction constructeur parente
function Animal(nom) {
    this.nom = nom;
}

// Ajout d'une méthode au prototype
Animal.prototype.direSalut = function() {
    console.log(`Salut, je suis ${this.nom}`);
};

// Fonction constructeur enfant
function Chien(nom, race) {
    Animal.call(this, nom);
    this.race = race;
}

// Héritage du prototype
Chien.prototype = Object.create(Animal.prototype);
Chien.prototype.constructor = Chien;

// Ajout d'une méthode spécifique au Chien
Chien.prototype.aboyer = function() {
    console.log("Woof! Woof!");
};

// Création d'une instance de Chien
let monChien = new Chien("Buddy", "Golden Retriever");

monChien.direSalut(); // Salut, je suis Buddy
monChien.aboyer(); // Woof! Woof!

6.2 Classes en JavaScript

À partir d'ECMAScript 2015 (ES6), JavaScript prend en charge les classes, qui simplifient la syntaxe pour la création de constructions similaires aux classes dans d'autres langages de programmation.

javascript
// Définition d'une classe
class Animal {
    constructor(nom) {
        this.nom = nom;
    }

    direSalut() {
        console.log(`Salut, je suis ${this.nom}`);
    }
}

// Héritage avec les classes
class Chien extends Animal {
    constructor(nom, race) {
        super(nom);
        this.race = race;
    }

    aboyer() {
        console.log("Woof! Woof!");
    }
}

// Création d'une instance de Chien
let monChien = new Chien("Buddy", "Golden Retriever");

monChien.direSalut(); // Salut, je suis Buddy
monChien.aboyer(); // Woof! Woof!

6.3 Propriétés statiques

Les propriétés statiques sont partagées par toutes les instances d'une classe et ne sont pas liées à une instance spécifique.

javascript
class Compteur {
    constructor() {
        // Incrémente la propriété statique à chaque nouvelle instance
        Compteur.instanceCount++;
    }

    // Méthode statique
    static getInstances() {
        return Compteur.instanceCount;
    }
}

// Initialisation de la propriété statique
Compteur.instanceCount = 0;

// Création de plusieurs instances
let compteur1 = new Compteur();
let compteur2 = new Compteur();

console.log(Compteur.getInstances()); // Affiche 2

6.4 Méthodes statiques

Les méthodes statiques sont des méthodes qui ne sont pas liées à une instance spécifique, mais à la classe elle-même.

javascript
class MathUtil {
    // Méthode statique pour additionner deux nombres
    static addition(a, b) {
        return a + b;
    }

    // Méthode statique pour multiplier deux nombres
    static multiplication(a, b) {
        return a * b;
    }
}

console.log(MathUtil.addition(5, 3)); // Affiche 8
console.log(MathUtil.multiplication(2, 4)); // Affiche 8

1. Introduction aux Tableaux en JavaScript

1.1 Définition des tableaux

Un tableau en JavaScript est une structure de données qui permet de stocker plusieurs valeurs dans une seule variable. Ces valeurs peuvent être de différents types et sont accessibles via des indices numériques.

1.2 Création de tableaux

Il existe plusieurs façons de créer des tableaux en JavaScript, mais la manière la plus courante est la notation littérale.

javascript
// Notation littérale de tableau
let fruits = ["pomme", "orange", "banane"];

Vous pouvez également utiliser le constructeur Array.

javascript
// Utilisation du constructeur Array
let nombres = new Array(1, 2, 3, 4, 5);

1.3 Accès aux éléments

Les éléments d'un tableau sont accessibles via leurs indices, qui commencent à 0 pour le premier élément.

javascript
let fruits = ["pomme", "orange", "banane"];

console.log(fruits[0]); // Affiche "pomme"
console.log(fruits[1]); // Affiche "orange"
console.log(fruits[2]); // Affiche "banane"

Vous pouvez également accéder aux éléments à partir de la fin du tableau en utilisant des indices négatifs.

javascript
console.log(fruits[-1]); // Affiche "banane" (dernier élément)
console.log(fruits[-2]); // Affiche "orange" (deuxième à partir de la fin)

2. Manipulation de Tableaux en JavaScript

2.1 Ajout et suppression d'éléments

Ajout d'éléments:
  • push(): Ajoute un ou plusieurs éléments à la fin du tableau.
javascript
let fruits = ["pomme", "orange", "banane"];
fruits.push("raisin");
console.log(fruits); // Affiche ["pomme", "orange", "banane", "raisin"]
  • unshift(): Ajoute un ou plusieurs éléments au début du tableau.
javascript
fruits.unshift("fraise");
console.log(fruits); // Affiche ["fraise", "pomme", "orange", "banane", "raisin"]
Suppression d'éléments:
  • pop(): Supprime le dernier élément du tableau.
javascript
fruits.pop();
console.log(fruits); // Affiche ["fraise", "pomme", "orange", "banane"]
  • shift(): Supprime le premier élément du tableau.
javascript
fruits.shift();
console.log(fruits); // Affiche ["pomme", "orange", "banane"]

2.2 Copie de tableaux

La copie d'un tableau est importante pour éviter la modification des tableaux d'origine. Voici quelques méthodes pour copier des tableaux :

  • slice(): Copie une partie du tableau.
javascript
let fruitsCopie = fruits.slice();
  • Spread Operator ...: Copie tous les éléments d'un tableau.
javascript
let fruitsCopie2 = [...fruits];
  • Array.from(): Crée une nouvelle instance d'Array à partir d'un objet itérable ou semblable à un tableau.
javascript
let fruitsCopie3 = Array.from(fruits);

2.3 Concaténation de tableaux

La concaténation de tableaux combine deux tableaux en un seul.

javascript
let legumes = ["carotte", "poireau", "brocoli"];

console.log(fruitsEtLegumes);
// Affiche ["pomme", "orange", "banane", "carotte", "poireau", "brocoli"]

Vous pouvez également utiliser le Spread Operator pour concaténer des tableaux.

javascript
let fruitsEtLegumes2 = [...fruits, ...legumes];

2.4 Recherche d'éléments

  • indexOf(): Retourne l'indice du premier élément trouvé dans le tableau, -1 s'il n'est pas présent.
javascript
let indiceOrange = fruits.indexOf("orange");
console.log(indiceOrange); // Affiche 1
  • includes(): Vérifie si un élément est présent dans le tableau, renvoie true ou false.
javascript
let contientBanane = fruits.includes("banane");
console.log(contientBanane); // Affiche true

4. Transformation de Tableaux en JavaScript

4.1 La méthode map()

La méthode map() crée un nouveau tableau en appelant une fonction fournie sur chaque élément du tableau d'origine.

javascript
let nombres = [1, 2, 3, 4];

// Doublons chaque élément du tableau
let doubles = nombres.map(function(nombre) {
    return nombre * 2;
});

console.log(doubles); // Affiche [2, 4, 6, 8]

La version avec une fonction fléchée :

javascript
let doubles = nombres.map(nombre => nombre * 2);

4.2 La méthode filter()

La méthode filter() crée un nouveau tableau avec tous les éléments du tableau d'origine qui satisfont une condition fournie par une fonction.

javascript
let nombres = [1, 2, 3, 4, 5];

// Filtrer les nombres pairs
let nombresPairs = nombres.filter(function(nombre) {
    return nombre % 2 === 0;
});

console.log(nombresPairs); // Affiche [2, 4]

Avec une fonction fléchée :

javascript
let nombresPairs = nombres.filter(nombre => nombre % 2 === 0);

4.3 La méthode reduce()

La méthode reduce() applique une fonction cumulative sur chaque élément du tableau pour réduire le tableau à une seule valeur.

javascript
let nombres = [1, 2, 3, 4, 5];

// Additionner tous les éléments du tableau
let somme = nombres.reduce(function(acc, nombre) {
    return acc + nombre;
}, 0);

console.log(somme); // Affiche 15

Avec une fonction fléchée :

javascript
let somme = nombres.reduce((acc, nombre) => acc + nombre, 0);

Dans cet exemple, acc représente l'accumulateur, et nombre représente chaque élément du tableau. La valeur initiale de l'accumulateur est spécifiée comme le deuxième argument de reduce() (dans cet exemple, c'est 0).

5. Tableaux Multidimensionnels en JavaScript

5.1 Création et accès

Un tableau multidimensionnel est un tableau qui contient d'autres tableaux. Cela crée une structure de tableau en deux dimensions ou plus.

javascript
// Création d'un tableau bidimensionnel
let tableau2D = [
    [1, 2, 3],

    [7, 8, 9]
];

console.log(tableau2D[0][0]); // Accès à l'élément à la première ligne, première colonne (affiche 1)
console.log(tableau2D[1][2]); // Accès à l'élément à la deuxième ligne, troisième colonne (affiche 6)

5.2 Manipulation

Vous pouvez manipuler les tableaux multidimensionnels de la même manière que les tableaux à une dimension, en utilisant des boucles, des méthodes de tableau, etc.

javascript
// Ajouter une nouvelle ligne
tableau2D.push([10, 11, 12]);

// Modifier un élément
tableau2D[1][1] = 55;

// Parcourir le tableau multidimensionnel
for (let i = 0; i < tableau2D.length; i++) {
    for (let j = 0; j < tableau2D[i].length; j++) {

La création et la manipulation de tableaux multidimensionnels sont utiles pour représenter des données tabulaires, des grilles, des matrices, etc.

6. Tableaux Avancés en JavaScript

6.1 Destructuration de Tableaux

La destructuration est une technique qui permet d'extraire des valeurs d'un tableau ou d'un objet et de les assigner à des variables.

javascript
let fruits = ["pomme", "orange", "banane"];

// Destructuration d'un tableau
let [premierFruit, deuxiemeFruit] = fruits;

console.log(premierFruit); // Affiche "pomme"
console.log(deuxiemeFruit); // Affiche "orange"

6.2 Méthodes Avancées (every, some, find, indexOf, etc.)

JavaScript propose plusieurs méthodes de tableau avancées pour effectuer des opérations complexes sur les tableaux.

  • every(): Vérifie si tous les éléments d'un tableau passent un test donné.
javascript
let nombres = [2, 4, 6, 8];

let tousPairs = nombres.every(function(nombre) {
    return nombre % 2 === 0;
});

console.log(tousPairs); // Affiche true
  • some(): Vérifie si au moins un élément d'un tableau passe un test donné.
javascript
let auMoinsUnPair = nombres.some(function(nombre) {
    return nombre % 2 === 0;
});

console.log(auMoinsUnPair); // Affiche true
  • find(): Renvoie la première valeur qui satisfait une condition donnée.
javascript
let nombrePair = nombres.find(function(nombre) {
    return nombre % 2 === 0;
});

console.log(nombrePair); // Affiche 2
  • indexOf(): Renvoie l'indice du premier élément correspondant dans le tableau, -1 s'il n'est pas trouvé.
javascript
let indexDe6 = nombres.indexOf(6);
console.log(indexDe6); // Affiche 2

6.3 Spread et Rest Operators avec les Tableaux

  • Spread Operator (...): Permet de copier les éléments d'un tableau dans un autre.
javascript
let nombres2 = [...nombres, 10, 12];
console.log(nombres2); // Affiche [2, 4, 6, 8, 10, 12]
  • Rest Operator (...): Permet de rassembler les arguments d'une fonction en un tableau.
javascript
function afficherNombres(...args) {
    console.log(args);
}

afficherNombres(1, 2, 3, 4, 5);
// Affiche [1, 2, 3, 4, 5]

  • indexOf() avec boucle pour rechercher toutes les occurrences d'une valeur.
javascript
function trouverToutesOccurrences(tableau, valeur) {
    let occurrences = [];
    let index = tableau.indexOf(valeur);

    while (index !== -1) {
        occurrences.push(index);
        index = tableau.indexOf(valeur, index + 1);
    }

    return occurrences;
}

8. Travail avec des Tableaux d'Objets en JavaScript

8.1 Filtrage et Tri d'Objets

  • Filtrage d'objets avec filter():

La méthode filter() peut être utilisée pour filtrer un tableau d'objets en fonction d'une condition spécifique.

javascript
let personnes = [
    { nom: "Alice", age: 25 },

let personnesAgeSuperieur25 = personnes.filter(function(personne) {

});

// Affiche [{ nom: "Bob", age: 30 }]
  • Tri d'objets avec sort():

La méthode sort() peut être utilisée pour trier un tableau d'objets en fonction d'une propriété spécifique.

javascript
// Trier les personnes par âge croissant
personnes.sort(function(a, b) {
    return a.age - b.age;
});

console.log(personnes);
// Affiche [{ nom: "Charlie", age: 22 }, { nom: "Alice", age: 25 }, { nom: "Bob", age: 30 }]
