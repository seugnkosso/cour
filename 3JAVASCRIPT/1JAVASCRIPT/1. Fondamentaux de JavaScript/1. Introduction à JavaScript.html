Introduction à JavaScript

Histoire de JavaScript

JavaScript a été créé par Brendan Eich en 1995, alors qu'il travaillait chez Netscape Communications Corporation. Initialement appelé Mocha, le langage a ensuite été renommé LiveScript avant de prendre son nom actuel, JavaScript, dans le but de capitaliser sur la popularité croissante du langage Java à l'époque.

JavaScript a été conçu pour être un langage de script côté client pour les navigateurs web, offrant aux développeurs la possibilité d'ajouter une interactivité dynamique aux pages web.

Rôle de JavaScript dans le développement web

JavaScript est essentiel au développement web moderne. Il est principalement utilisé côté client pour rendre les pages web interactives et dynamiques. Voici quelques-uns des rôles clés de JavaScript dans le développement web :

  1. Interactivité utilisateur : JavaScript permet de créer des interfaces utilisateur interactives en répondant aux actions de l'utilisateur sur une page web.

  2. Manipulation du DOM : Le Document Object Model (DOM) représente la structure d'une page web. JavaScript peut être utilisé pour manipuler le DOM, ce qui signifie qu'il peut ajouter, supprimer ou modifier des éléments sur une page.

  3. Requêtes asynchrones : JavaScript facilite les requêtes asynchrones vers des serveurs, ce qui permet de mettre à jour des parties spécifiques d'une page sans recharger toute la page. Cela est couramment utilisé dans les applications web modernes pour améliorer les performances.

  4. Contrôle des événements : JavaScript permet de gérer les événements tels que les clics de souris, les saisies clavier, etc., pour déclencher des actions spécifiques.

  5. Validation des données côté client : JavaScript peut être utilisé pour valider les données côté client avant leur envoi à un serveur, offrant ainsi une meilleure expérience utilisateur.

Concepts de base en JavaScript

Variables

En JavaScript, les variables sont utilisées pour stocker des valeurs. Vous pouvez déclarer une variable en utilisant les mots-clés var, let, ou const. Par exemple :

javascript
var x = 5;   // Ancienne façon de déclarer une variable (évitez d'utiliser var)
let y = 10;  // Variable pouvant être réaffectée
const z = 15; // Constante, dont la valeur ne peut pas être modifiée

Types de données

JavaScript est un langage dynamiquement typé, ce qui signifie que le type d'une variable peut changer dynamiquement. Les types de données de base incluent :

  • Number pour les nombres (entiers et décimaux)
  • String pour les chaînes de caractères
  • Boolean pour les valeurs booléennes (true ou false)
  • Array pour les tableaux
  • Object pour les objets
  • null et undefined pour les valeurs nulles ou non définies

Opérateurs

Les opérateurs en JavaScript sont utilisés pour effectuer des opérations sur des variables et des valeurs. Les opérateurs incluent les opérateurs arithmétiques (+, -, *, /), les opérateurs de comparaison (==, ===, !=, !==), les opérateurs logiques (&&, ||, !), etc.

javascript
let a = 5;
let b = 10;

let sum = a + b;     // Addition
let difference = a - b;  // Soustraction
let product = a * b;  // Multiplication
let quotient = a / b;  // Division

let isEqual = (a === b);  // Comparaison stricte (vérifie également le type)
let logicalAnd = (a > 0 && b < 20);  // Opérateur logique ET
