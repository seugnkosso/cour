1. Introduction à la méthode substring()

La méthode substring() est une fonction intégrée en JavaScript qui permet d'extraire une sous-chaîne d'une chaîne de caractères. Elle prend en compte deux indices en tant que paramètres, représentant le début et la fin de la sous-chaîne à extraire.

2. Syntaxe de base

La syntaxe de base de la méthode substring() est la suivante :

javascript
string.substring(startIndex, endIndex)
  • startIndex : L'indice du premier caractère de la sous-chaîne (inclus).
  • endIndex : L'indice du caractère après le dernier caractère de la sous-chaîne (non inclus).

3. Utilisation avec des indices positifs

Lorsque des indices positifs sont utilisés, la méthode substring() extrait les caractères entre les indices spécifiés.

javascript
let originalString = "Exemple de chaîne";
let subStringResult = originalString.substring(8, 14);

console.log(subStringResult); // Affiche "de cha"

Dans cet exemple, la sous-chaîne extraite commence à l'indice 8 (inclus) et se termine à l'indice 14 (non inclus).

4. Utilisation avec des indices négatifs

La méthode substring() prend également en charge les indices négatifs. Les indices négatifs comptent à partir de la fin de la chaîne, où -1 représente le dernier caractère.

javascript
let originalString = "Exemple de chaîne";
let subStringResult = originalString.substring(-6, -1);

console.log(subStringResult); // Affiche "chaîn"

Dans cet exemple, la sous-chaîne extraite commence au sixième caractère en partant de la fin et se termine avant le dernier caractère.

5. Exemples pratiques

5.1 Extraction d'une sous-chaîne sans spécifier l'indice de fin

Si l'indice de fin n'est pas spécifié, la méthode substring() extrait les caractères jusqu'à la fin de la chaîne.

javascript
let originalString = "Exemple de chaîne";
let subStringResult = originalString.substring(8);

console.log(subStringResult); // Affiche "de chaîne"

Dans cet exemple, la sous-chaîne extraite commence à l'indice 8 et se prolonge jusqu'à la fin de la chaîne.

5.2 Utilisation avec des indices dynamiques

Les indices peuvent également être des variables, ce qui permet une utilisation dynamique de la méthode substring().

javascript
let originalString = "Exemple de chaîne";
let startIndex = 8;
let endIndex = 14;
let subStringResult = originalString.substring(startIndex, endIndex);

console.log(subStringResult); // Affiche "de cha"

En résumé, la méthode substring() est utile pour extraire des sous-chaînes à partir d'une chaîne de caractères en spécifiant les indices de début et de fin. Elle prend en charge à la fois les indices positifs et négatifs, offrant ainsi une flexibilité dans la manipulation des chaînes de caractères en JavaScript.

1. Introduction à la méthode slice()

La méthode slice() est une fonction intégrée en JavaScript qui permet d'extraire une portion d'une chaîne de caractères ou d'un tableau. Elle prend en compte un ou deux indices en tant que paramètres, représentant le début et la fin de la portion à extraire.

2. Syntaxe de base

La syntaxe de base de la méthode slice() est la suivante :

javascript
string.slice(startIndex, endIndex)
  • startIndex : L'indice du premier élément de la portion (inclus).
  • endIndex : L'indice du premier élément à exclure de la portion (non inclus).

3. Utilisation avec des indices positifs

Lorsque des indices positifs sont utilisés, la méthode slice() extrait les éléments entre les indices spécifiés.

javascript
let originalString = "Exemple de chaîne";
let slicedResult = originalString.slice(8, 14);

console.log(slicedResult); // Affiche "de cha"

Dans cet exemple, la portion extraite commence à l'indice 8 (inclus) et se termine à l'indice 14 (non inclus).

4. Utilisation avec des indices négatifs

La méthode slice() prend également en charge les indices négatifs, où -1 représente le dernier élément.

javascript
let originalString = "Exemple de chaîne";
let slicedResult = originalString.slice(-6, -1);

console.log(slicedResult); // Affiche "chaîn"

Dans cet exemple, la portion extraite commence au sixième élément en partant de la fin et se termine avant le dernier élément.

5. Utilisation avec un seul paramètre

Si un seul paramètre est fourni à la méthode slice(), elle extrait les éléments à partir de l'indice spécifié jusqu'à la fin de la chaîne.

javascript
let originalString = "Exemple de chaîne";
let slicedResult = originalString.slice(8);

console.log(slicedResult); // Affiche "de chaîne"

Dans cet exemple, la portion extraite commence à l'indice 8 et se prolonge jusqu'à la fin de la chaîne.

6. Exemples pratiques

6.1 Extraction d'une sous-chaîne avec slice()

javascript
let originalString = "Bienvenue sur JavaScript";
let extractedSubstring = originalString.slice(12, 22);

console.log(extractedSubstring); // Affiche "JavaScript"

6.2 Utilisation avec des indices dynamiques

javascript
let originalString = "Manipulation de chaînes";
let startIndex = 5;
let endIndex = 14;
let slicedResult = originalString.slice(startIndex, endIndex);

console.log(slicedResult); // Affiche "pulation d"

La méthode slice() offre une flexibilité similaire à substring() pour extraire des portions de chaînes de caractères en JavaScript. Elle peut être utilisée avec des indices positifs, négatifs, ou même avec un seul paramètre, offrant ainsi des options variées pour la manipulation de chaînes de caractères.

1. Introduction à la méthode substr()

La méthode substr() est une fonction intégrée en JavaScript qui permet d'extraire une sous-chaîne d'une chaîne de caractères. Elle prend en compte deux paramètres : l'indice de départ et la longueur de la sous-chaîne à extraire.

2. Syntaxe de base

La syntaxe de base de la méthode substr() est la suivante :

javascript
string.substr(startIndex, length)
  • startIndex : L'indice du premier caractère de la sous-chaîne à extraire.
  • length : La longueur de la sous-chaîne à extraire.

3. Utilisation avec des indices positifs

Lorsque des indices positifs sont utilisés, la méthode substr() extrait une sous-chaîne en commençant par l'indice spécifié et en prenant en compte la longueur spécifiée.

javascript
let originalString = "Exemple de chaîne";
let subStrResult = originalString.substr(8, 4);

console.log(subStrResult); // Affiche "de c"

Dans cet exemple, la sous-chaîne extraite commence à l'indice 8 et a une longueur de 4 caractères.

4. Utilisation avec des indices négatifs

La méthode substr() ne prend pas en charge les indices négatifs. Si un indice négatif est fourni, il sera traité comme si l'indice était 0.

5. Exemples pratiques

5.1 Extraction d'une sous-chaîne avec substr()

javascript
let originalString = "Bienvenue sur JavaScript";
let extractedSubstring = originalString.substr(12, 10);

console.log(extractedSubstring); // Affiche "JavaScript"

Dans cet exemple, la sous-chaîne extraite commence à l'indice 12 et a une longueur de 10 caractères.

5.2 Utilisation avec des indices dynamiques

javascript
let originalString = "Manipulation de chaînes";
let startIndex = 5;
let length = 8;
let subStrResult = originalString.substr(startIndex, length);

console.log(subStrResult); // Affiche "ulatio"

La méthode substr() est particulièrement utile lorsque vous connaissez l'indice de départ et la longueur de la sous-chaîne que vous souhaitez extraire.

En résumé, la méthode substr() est une option efficace pour extraire des sous-chaînes de longueur spécifique à partir d'une chaîne de caractères en JavaScript. Elle offre une alternative aux méthodes substring() et slice(), en se concentrant sur la spécification de la longueur plutôt que sur l'indice de fin.

1. Similitudes entre substring(), slice(), et substr()

1.1. Objectif commun

  • Les trois méthodes (substring(), slice(), et substr()) sont utilisées pour extraire des sous-chaînes à partir d'une chaîne de caractères en JavaScript.

1.2. Paramètres d'indices

  • Elles utilisent tous des indices pour déterminer la portion de la chaîne à extraire.

2. Différences entre substring(), slice(), et substr()

2.1. Paramètres

  • substring() : Prend deux indices (début et fin).
  • slice() : Prend deux indices (début et fin) ou un seul indice (début) pour extraire jusqu'à la fin.
  • substr() : Prend un indice de départ et une longueur.

2.2. Gestion des indices négatifs

  • substring() : Permet l'utilisation d'indices négatifs.
  • slice() : Permet l'utilisation d'indices négatifs pour compter à partir de la fin.
  • substr() : Ne prend pas en charge les indices négatifs.

2.3. Comportement avec indices hors limites

  • substring() : Si l'indice de fin est inférieur à l'indice de début, les deux sont inversés.
  • slice() : Si l'indice de fin est inférieur à l'indice de début, la portion extraite sera vide.
  • substr() : Si l'indice de début est hors limites, il est ajusté au début ou à la fin de la chaîne.

2.4. Flexibilité

  • substring() : Moins flexible car elle ne permet pas de spécifier la longueur directement.
  • slice() : Offre plus de flexibilité en permettant de spécifier la longueur ou de ne fournir qu'un indice pour extraire jusqu'à la fin.
  • substr() : Spécifie la longueur plutôt que l'indice de fin, ce qui peut être plus intuitif dans certains cas.

3. Cas d'utilisation appropriés

3.1. Choix basé sur les indices

  • Si vous avez besoin de spécifier deux indices pour délimiter la sous-chaîne, substring() ou slice() peuvent être utilisées.

3.2. Choix basé sur la longueur

  • Si vous avez la longueur de la sous-chaîne souhaitée, substr() offre une syntaxe plus claire.

3.3. Indices négatifs

  • Si vous avez besoin d'utiliser des indices négatifs, slice() est la seule option parmi les trois.

3.4. Flexibilité

  • Si vous avez besoin de flexibilité pour extraire jusqu'à la fin ou si vous travaillez avec des indices négatifs, slice() peut être plus appropriée.

En conclusion, le choix entre substring(), slice(), et substr() dépend des besoins spécifiques de votre scénario. Chacune a ses propres avantages et inconvénients, et la décision peut être basée sur la facilité d'utilisation, la flexibilité requise et la clarté du code dans un contexte particulier.

1. Gestion des indices hors limites

1.1 Vérification des indices avant utilisation

Lorsque vous travaillez avec des indices pour extraire des sous-chaînes, il est important de vérifier qu'ils se trouvent dans les limites de la chaîne. Vous pouvez utiliser des conditions pour éviter les erreurs.

Exemple :

javascript
let myString = "Exemple de chaîne";
let startIndex = 8;
let endIndex = 14;

if (startIndex >= 0 && endIndex <= myString.length && startIndex < endIndex) {
  let subString = myString.substring(startIndex, endIndex);
  console.log(subString);
} else {
  console.log("Indices hors limites.");
}

1.2 Utilisation de valeurs par défaut

Si vous utilisez des indices fournis par l'utilisateur ou des valeurs dynamiques, pensez à définir des valeurs par défaut pour éviter des comportements inattendus.

Exemple :

javascript
function extractSubString(myString, startIndex = 0, endIndex = myString.length) {
  if (startIndex >= 0 && endIndex <= myString.length && startIndex < endIndex) {
    return myString.substring(startIndex, endIndex);
  } else {
    console.log("Indices hors limites. Utilisation de la chaîne complète.");
    return myString;
  }
}

let result = extractSubString("Exemple de chaîne", 8, 14);
console.log(result);

  let cleanedInput = input.replace(/\D/g, '');
