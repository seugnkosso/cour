<body>

  </div>

    // Sélection de l'élément enfant
    var childElement = document.getElementById("child");

    // Utilisation de parentNode pour accéder à l'élément parent
    var parentElement = childElement.parentNode;

    // Affichage dans la console pour vérification
    console.log(parentElement);
  </script>

</body>
</html>

Dans cet exemple, childElement est l'élément enfant que nous avons sélectionné par son ID, puis nous utilisons parentNode pour accéder à son parent (parentElement). Vous pouvez tester cela dans votre navigateur et vérifier la console pour voir l'élément parent.

2. Gestion des espaces vides et du type de nœud avec childNodes

La NodeList retournée par childNodes peut contenir des nœuds de texte représentant des espaces vides entre les éléments. De plus, elle peut contenir différents types de nœuds tels que des éléments, des commentaires, des nœuds de texte, etc. Voici comment vous pouvez gérer cela :

html
<script>
  var parentElement = document.getElementById("parent");
  var childNodes = parentElement.childNodes;

  // Parcourir les nœuds enfants
  for (var i = 0; i < childNodes.length; i++) {
    var node = childNodes[i];

    // Vérifier si le nœud est un élément
    if (node.nodeType === 1) { // 1 correspond à un élément
      console.log("Element:", node.nodeName);
    }

    // Vérifier si le nœud est un nœud de texte non vide
    if (node.nodeType === 3 && node.nodeValue.trim() !== "") { // 3 correspond à un nœud de texte
      console.log("Texte:", node.nodeValue);
    }
  }
</script>

Dans cet exemple, nous parcourons les nœuds enfants et utilisons nodeType pour vérifier le type de nœud. Le type 1 correspond à un élément, et le type 3 correspond à un nœud de texte. Nous vérifions également si le nœud de texte n'est pas vide en utilisant nodeValue.trim().

    if (node.childNodes.length > 0) {
