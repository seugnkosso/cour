A. Rôle du Contrôleur dans la gestion des événements

Dans le modèle MVC, le contrôleur est responsable de traiter les entrées de l'utilisateur, généralement sous forme d'événements, et de mettre à jour le modèle et la vue en conséquence. Voici comment vous pouvez gérer les événements dans le contrôleur :

1. Création du Contrôleur

Commencez par créer votre contrôleur. Dans le contexte de Vue.js, vous pouvez utiliser un objet Vue pour cela :

javascript
// controller.js
const app = new Vue({
  el: '#app', // l'élément DOM associé au contrôleur
  data: {
    // les données du modèle
    message: 'Bonjour, Monde!'
  },
  methods: {
    // les méthodes du contrôleur
    handleClick: function () {
      this.message = 'Le bouton a été cliqué!';
    }
  }
});

B. Liaison entre la Vue et le Contrôleur

Pour que la vue et le contrôleur soient liés, vous devez associer les éléments de la vue aux méthodes du contrôleur. Cela se fait généralement dans le fichier HTML où vous définissez votre vue.

1. Création de la Vue

html
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Vue en MVC - Gestion des Événements</title>
</head>
<body>
  <div id="app">
    <p>{{ message }}</p>
    <button @click="handleClick">Cliquer ici</button>
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
  <script src="controller.js"></script>
</body>
</html>

Dans cet exemple, le bouton est associé à la méthode handleClick du contrôleur à l'aide de la directive @click.

C. Implémentation de méthodes de gestion d'événements dans le Contrôleur

Maintenant, implémentez la méthode de gestion d'événements dans le contrôleur. La méthode handleClick sera appelée lorsque le bouton est cliqué, mettant à jour le modèle et la vue.

javascript
// controller.js
const app = new Vue({

    message: 'Bonjour, Monde!'

A. Exemples de gestion d'événements pour différents types d'application

1. Application de To-Do List

a. Vue

javascript
// todoApp.js
const todoApp = new Vue({
  el: '#todoApp',
  data: {
    todos: [],
    newTodo: ''
  },
  methods: {
    addTodo: function () {
      this.todos.push({ text: this.newTodo, completed: false });
      this.newTodo = '';
    },
    completeTodo: function (index) {
      this.todos[index].completed = true;
    }
  }
});

b. HTML

html
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Vue en MVC - Exemple To-Do List</title>
</head>
<body>
  <div id="todoApp">
    <ul>
      <li v-for="(todo, index) in todos" :key="index">
        {{ todo.text }}
        <button @click="completeTodo(index)" v-if="!todo.completed">Terminé</button>
      </li>
    </ul>
    <input v-model="newTodo" @keyup.enter="addTodo" placeholder="Nouvelle tâche">
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
  <script src="todoApp.js"></script>
</body>
</html>

Dans cet exemple, la Vue gère l'ajout de nouvelles tâches et la complétion des tâches existantes dans une liste de tâches.

1. Application de Chat en Temps Réel

a. Vue

javascript
// chatApp.js
const chatApp = new Vue({
  el: '#chatApp',
  data: {
    messages: [],
    newMessage: ''
  },
  methods: {
    sendMessage: function () {
      // Logique pour envoyer le message au serveur en temps réel
      this.messages.push({ text: this.newMessage, user: 'Moi' });
      this.newMessage = '';
    }
  }
});

b. HTML

html
<!-- index.html -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">

  <title>Vue en MVC - Exemple Chat en Temps Réel</title>
</head>
<body>
  <div id="chatApp">
    <div v-for="(message, index) in messages" :key="index">
      <strong>{{ message.user }}:</strong> {{ message.text }}

    <input v-model="newMessage" @keyup.enter="sendMessage" placeholder="Votre message">
  </div>

  <script src="https://cdn.jsdelivr.net/npm/vue@2"></script>
  <script src="chatApp.js"></script>
</body>
</html>

Dans cet exemple, la Vue gère l'envoi de messages en temps réel dans une application de chat.
