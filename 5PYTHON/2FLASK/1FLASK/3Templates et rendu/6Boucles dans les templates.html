1. Syntaxe de base des boucles {% for %} dans les templates Flask :

La syntaxe de base d'une boucle {% for %} dans un template Flask est la suivante :

html
{% for element in iterable %}
    <!-- contenu à répéter pour chaque élément de l'itérable -->
{% endfor %}
  • element est la variable qui contiendra chaque élément de l'itérable à chaque itération de la boucle.
  • iterable est l'objet itérable, comme une liste, un tuple, ou même un dictionnaire, sur lequel vous souhaitez itérer.

2. Exemples pratiques d'utilisation des boucles {% for %} pour itérer sur des données :

a. Itération sur une liste :

python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    fruits = ['pomme', 'banane', 'orange']
    return render_template('index.html', fruits=fruits)

# Dans le template 'index.html'
# Iteration sur la liste fruits
<ul>
    {% for fruit in fruits %}
        <li>{{ fruit }}</li>
    {% endfor %}
</ul>

b. Itération sur un dictionnaire :

python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    fruits_stock = {'pomme': 10, 'banane': 5, 'orange': 8}
    return render_template('index.html', fruits_stock=fruits_stock)

# Dans le template 'index.html'
# Iteration sur les clés et valeurs du dictionnaire fruits_stock
<ul>
    {% for fruit, stock in fruits_stock.items() %}
        <li>{{ fruit }} - Stock: {{ stock }}</li>
    {% endfor %}
</ul>

c. Itération sur une plage de nombres :

python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    return render_template('index.html', range_value=range(1, 6))

# Dans le template 'index.html'
# Iteration sur une plage de nombres de 1 à 5
<ul>
    {% for num in range_value %}
        <li>{{ num }}</li>
    {% endfor %}
</ul>

Ces exemples illustrent les bases de l'utilisation des boucles {% for %} dans les templates Flask. Vous pouvez les adapter et les étendre en fonction des besoins spécifiques de votre application.

Itération sur des listes dans les templates Flask :

Dans Flask, vous pouvez facilement itérer sur des listes dans les templates à l'aide de la syntaxe {% for item in liste %}. Cette syntaxe vous permet de parcourir chaque élément de la liste et d'effectuer des actions en conséquence.

Illustration de l'itération sur des listes de différentes tailles :

Considérons d'abord une liste de fruits :

python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    fruits = ['pomme', 'banane', 'orange']
    return render_template('index.html', fruits=fruits)

Utilisation de la syntaxe {% for item in liste %} pour parcourir les éléments de la liste :

Dans votre template HTML (index.html), vous pouvez utiliser la syntaxe {% for item in liste %} pour itérer sur les éléments de la liste fruits :

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Liste de Fruits</title>
</head>
<body>
    <h1>Liste de Fruits</h1>
    <ul>
        {% for fruit in fruits %}
            <li>{{ fruit }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Dans cet exemple, {% for fruit in fruits %} parcourt chaque élément de la liste fruits et utilise la variable fruit pour représenter chaque élément. À chaque itération, le contenu entre {% for %} et {% endfor %} est répété avec la variable fruit représentant l'élément actuel.

Résultat :

Lorsque vous accédez à votre application Flask dans le navigateur, vous verrez la liste des fruits affichée sous forme de liste non ordonnée.

  • Pomme
  • Banane
  • Orange

Cela montre comment itérer sur une liste dans un template Flask en utilisant la syntaxe {% for item in liste %} et en illustrant l'itération sur des listes de différentes tailles.

Itération sur des dictionnaires dans les templates Flask :

Dans Flask, itérer sur des dictionnaires dans les templates est assez similaire à l'itération sur des listes, mais avec une légère différence de syntaxe. Vous pouvez parcourir à la fois les clés et les valeurs d'un dictionnaire.

Explication de la manière de parcourir les clés et les valeurs d'un dictionnaire :

Pour parcourir à la fois les clés et les valeurs d'un dictionnaire dans un template Flask, vous pouvez utiliser la méthode items() du dictionnaire. Cela vous permet de récupérer à la fois la clé et la valeur à chaque itération de la boucle.

Utilisation de la syntaxe {% for key, value in dictionnaire.items() %} pour itérer sur un dictionnaire :

Voici un exemple illustrant comment parcourir un dictionnaire de fruits avec leurs quantités correspondantes :

python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    fruits_stock = {'pomme': 10, 'banane': 5, 'orange': 8}
    return render_template('index.html', fruits_stock=fruits_stock)

Dans le template HTML (index.html), utilisez la syntaxe {% for key, value in dictionnaire.items() %} pour parcourir les clés et les valeurs du dictionnaire fruits_stock :

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Stock de Fruits</title>
</head>
<body>
    <h1>Stock de Fruits</h1>
    <ul>
        {% for fruit, quantity in fruits_stock.items() %}
            <li>{{ fruit }} - Quantité: {{ quantity }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Dans cet exemple, {% for fruit, quantity in fruits_stock.items() %} parcourt chaque paire clé-valeur du dictionnaire fruits_stock, avec fruit représentant la clé (nom du fruit) et quantity représentant la valeur (quantité de ce fruit). À chaque itération, le contenu entre {% for %} et {% endfor %} est répété avec les variables fruit et quantity représentant la clé et la valeur actuelles respectivement.

Résultat :

Lorsque vous accédez à votre application Flask dans le navigateur, vous verrez la liste des fruits avec leurs quantités affichées.

  • Pomme - Quantité: 10
  • Banane - Quantité: 5
  • Orange - Quantité: 8

Cela montre comment itérer sur un dictionnaire dans un template Flask en utilisant la syntaxe {% for key, value in dictionnaire.items() %} et en parcourant à la fois les clés et les valeurs du dictionnaire.

L'itération sur des objets personnalisés dans les templates Flask peut être un peu plus complexe que l'itération sur des listes ou des dictionnaires, car vous devez vous assurer que les attributs de l'objet sont accessibles dans votre template. Voici un tutoriel complet sur la façon d'itérer sur des objets dans les templates Flask :

Introduction à l'itération sur des objets personnalisés passés aux templates :

Lorsque vous souhaitez itérer sur des objets personnalisés dans vos templates Flask, vous devez vous assurer que ces objets sont correctement passés aux templates à partir de vos vues Flask. Cela peut nécessiter une manipulation appropriée des données dans vos vues pour garantir que les attributs nécessaires de l'objet sont accessibles dans le template.

Utilisation de l'attribut spécifique de l'objet pour extraire les données à afficher :

Supposons que vous ayez une classe Fruit avec des attributs tels que nom et quantite, et que vous souhaitiez itérer sur une liste d'objets Fruit dans votre template. Voici comment vous pourriez le faire :

python
from flask import Flask, render_template

app = Flask(__name__)

# Définition de la classe Fruit
class Fruit:
    def __init__(self, nom, quantite):
        self.nom = nom
        self.quantite = quantite

# Route de l'application Flask
@app.route('/')
def index():
    fruits = [Fruit('pomme', 10), Fruit('banane', 5), Fruit('orange', 8)]
    return render_template('index.html', fruits=fruits)

Dans cet exemple, nous avons une classe Fruit avec des attributs nom et quantite. Nous créons ensuite une liste d'objets Fruit que nous passons à notre template.

Dans votre template HTML (index.html), vous pouvez utiliser la syntaxe normale de Jinja2 pour itérer sur les objets Fruit et extraire les données pertinentes :

html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Liste de Fruits</title>
</head>
<body>
    <h1>Liste de Fruits</h1>
    <ul>
        {% for fruit in fruits %}
            <li>{{ fruit.nom }} - Quantité: {{ fruit.quantite }}</li>
        {% endfor %}
    </ul>
</body>
</html>

Dans cet exemple, {% for fruit in fruits %} parcourt chaque objet Fruit dans la liste fruits, et {% fruit.nom %} et {% fruit.quantite %} extraient respectivement le nom et la quantité de chaque objet Fruit.

Résultat :

Lorsque vous accédez à votre application Flask dans le navigateur, vous verrez la liste des fruits avec leurs quantités affichées.

  • Pomme - Quantité: 10
  • Banane - Quantité: 5
  • Orange - Quantité: 8

Cela montre comment itérer sur des objets dans un template Flask en passant des objets personnalisés aux templates et en extrayant les données nécessaires à partir de ces objets.

Les filtres Jinja2 offrent une manière puissante de manipuler et de formater les données affichées dans les templates Flask. Voici un tutoriel complet sur l'utilisation des filtres Jinja2 dans les boucles :

Présentation des filtres Jinja2 pour modifier les données affichées dans les boucles :

Les filtres Jinja2 sont des outils puissants pour transformer les données avant de les afficher dans les templates Flask. Ils peuvent être utilisés pour effectuer diverses opérations telles que la mise en majuscule ou en minuscule, le formatage de dates, la découpe de chaînes, etc.

Exemples d'utilisation de filtres tels que upper(), lower(), slice(), etc. :

Voici quelques exemples d'utilisation courante des filtres Jinja2 dans les boucles :

1. Utilisation du filtre upper() pour mettre en majuscule une chaîne :

html
{{ "hello" | upper }}

Résultat : HELLO

2. Utilisation du filtre lower() pour mettre en minuscule une chaîne :

html
{{ "WORLD" | lower }}

Résultat : world

3. Utilisation du filtre slice() pour extraire une sous-chaîne :

html
{{ "abcdefghij" | slice(2, 5) }}

Résultat : cde

4. Utilisation du filtre default() pour définir une valeur par défaut si la variable est non définie :

html
{{ my_variable | default("valeur par défaut") }}

5. Utilisation du filtre date() pour formater une date :

html
{{ my_date | date("Y-m-d") }}

6. Utilisation du filtre length() pour obtenir la longueur d'une liste ou d'une chaîne :

html
{{ my_list | length }}
{{ my_string | length }}

7. Utilisation du filtre join() pour joindre les éléments d'une liste en une chaîne :

html
{{ my_list | join(", ") }}

Ces exemples illustrent comment les filtres Jinja2 peuvent être utilisés pour manipuler les données affichées dans les templates Flask. Vous pouvez les utiliser de manière créative pour formater et transformer les données selon vos besoins spécifiques.
