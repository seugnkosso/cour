Bien sûr, voici un guide détaillé pour configurer les paramètres réseau, y compris les paramètres IP (statique ou dynamique) et la configuration DNS sur un serveur Windows.

Étape 1 : Accéder aux paramètres réseau

  1. Appuyez sur les touches Windows + X pour ouvrir le menu d'options.
  2. Cliquez sur "Connexions réseau" ou "Connexions réseau et Internet" selon la version de Windows Server.

Étape 2 : Accéder aux propriétés de la connexion

  1. Trouvez la connexion réseau que vous souhaitez configurer (par exemple, Ethernet) et faites un clic droit dessus.
  2. Sélectionnez "Propriétés" dans le menu contextuel.

Étape 3 : Configurer l'adresse IP (Statique)

Pour une adresse IP statique :

  1. Sélectionnez "Protocole Internet version 4 (TCP/IPv4)" dans la liste et cliquez sur "Propriétés".

  2. Cochez l'option "Utiliser l'adresse IP suivante" et entrez l'adresse IP, le masque de sous-réseau, et la passerelle par défaut appropriés.

  3. Entrez les serveurs DNS préférés et auxiliaires (si vous les connaissez). Cliquez sur "OK" pour appliquer les modifications.

Pour une adresse IP dynamique (DHCP) :

  1. Choisissez l'option "Obtenir une adresse IP automatiquement" et "Obtenir les adresses des serveurs DNS automatiquement".

  2. Cliquez sur "OK" pour appliquer les modifications.

Étape 4 : Configurer le DNS

  1. Toujours dans les propriétés de la connexion, cliquez sur "Protocole Internet version 4 (TCP/IPv4)" et sélectionnez "Propriétés".

  2. Si vous utilisez des serveurs DNS spécifiques, cochez "Utiliser les adresses de serveurs DNS suivantes" et entrez les adresses DNS préférées et auxiliaires.

  3. Cliquez sur "OK" pour appliquer les modifications.

Étape 5 : Vérifier les paramètres réseau

  1. Ouvrez une invite de commande (Cmd) en tant qu'administrateur.

  2. Tapez ipconfig /all pour afficher toutes les informations réseau, y compris les adresses IP et les serveurs DNS.

Assurez-vous que les informations affichées correspondent à celles que vous avez configurées.

C'est tout ! Vous avez maintenant configuré avec succès les paramètres réseau, y compris l'adresse IP (statique ou dynamique) et les serveurs DNS sur votre serveur Windows. N'oubliez pas de tester la connectivité réseau pour vous assurer que tout fonctionne correctement.
