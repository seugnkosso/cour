Tutoriel : Création de Comptes Utilisateurs sur Windows Server

Étape 1 : Accès au Gestionnaire de serveur

  1. Connectez-vous au serveur Windows avec un compte administrateur.
  2. Cliquez sur le bouton "Démarrer" dans le coin inférieur gauche.
  3. Sélectionnez "Gestionnaire de serveur" dans la liste des applications.

Étape 2 : Ouverture du Gestionnaire de serveur

  1. Dans le Gestionnaire de serveur, vous trouverez plusieurs options. Cliquez sur "Outils" dans le coin supérieur droit.
  2. Sélectionnez "Utilisateurs et ordinateurs Active Directory" dans la liste des outils.

Étape 3 : Navigation dans l'Active Directory

  1. Une fois dans "Utilisateurs et ordinateurs Active Directory", vous verrez la structure de votre domaine Active Directory à gauche.
  2. Naviguez jusqu'à l'endroit où vous souhaitez créer le nouvel utilisateur. Cela pourrait être l'unité d'organisation (OU) appropriée ou le conteneur "Utilisateurs" par défaut.

Étape 4 : Création du Compte Utilisateur

  1. Faites un clic droit sur l'emplacement où vous souhaitez créer le nouvel utilisateur.
  2. Sélectionnez "Nouveau" puis "Utilisateur" dans le menu contextuel.

Étape 5 : Assistant de création de compte utilisateur

  1. L'Assistant de création de compte utilisateur s'ouvrira. Suivez les étapes de l'assistant pour spécifier les informations du nouvel utilisateur.
    • Prénom et nom : Entrez le prénom et le nom de l'utilisateur.
    • Nom d'ouverture de session : Un nom d'ouverture de session unique pour l'utilisateur.
    • Mot de passe : Définissez un mot de passe pour le compte utilisateur.
    • Vous pouvez également définir d'autres propriétés telles que le groupe auquel appartient l'utilisateur.

Étape 6 : Résumé et Création

  1. Une fois toutes les informations fournies, l'assistant affichera un résumé. Vérifiez les détails.
  2. Cliquez sur "Terminer" pour créer le compte utilisateur.

Étape 7 : Vérification du Compte Utilisateur

  1. Retournez à "Utilisateurs et ordinateurs Active Directory".
  2. Recherchez le nouvel utilisateur que vous venez de créer.
  3. Confirmez que le compte utilisateur apparaît correctement dans la structure Active Directory.

Félicitations ! Vous avez créé avec succès un compte utilisateur sur votre serveur Windows à l'aide du Gestionnaire de serveur et d'Active Directory. Vous pouvez désormais utiliser ce compte pour l'authentification sur le réseau.

Tutoriel : Modification des Propriétés des Comptes Utilisateurs sur Windows Server

Étape 1 : Accéder aux Propriétés d'un Compte Utilisateur

  1. Ouvrez le "Gestionnaire de serveur" sur votre serveur Windows.
  2. Accédez à "Outils" > "Utilisateurs et ordinateurs Active Directory".

Étape 2 : Trouver le Compte Utilisateur

  1. Naviguez dans la structure Active Directory pour localiser le compte utilisateur que vous souhaitez modifier.
  2. Faites un clic droit sur le compte utilisateur et sélectionnez "Propriétés".

Étape 3 : Informations Générales

  1. Dans l'onglet "Général", ajustez les informations telles que le prénom, le nom, le nom d'ouverture de session, etc.
  2. Si nécessaire, modifiez le mot de passe dans l'onglet "Compte".

Étape 4 : Configuration des Onglets Principaux

  1. Onglet "Compte" :

    • Spécifiez les paramètres liés au compte, tels que la date d'expiration du compte, les restrictions de connexion, etc.

  2. Onglet "Profil" :

    • Configurez le chemin du profil utilisateur, le script de connexion, le lecteur de démarrage, etc.

  3. Onglet "Membre de" :

    • Ajoutez ou supprimez l'utilisateur des groupes nécessaires.

  4. Onglet "Téléphone" (le cas échéant) :

    • Ajoutez ou modifiez les informations liées au téléphone de l'utilisateur.

Étape 5 : Définir les Droits et Permissions

  1. Onglet "Autorisations" :

    • Gérez les droits et autorisations de l'utilisateur sur des ressources spécifiques.

  2. Onglet "Contrôle total" (le cas échéant) :

    • Configurez les contrôles totaux sur des objets spécifiques pour l'utilisateur.

Étape 6 : Appliquer les Modifications

  1. Une fois toutes les modifications effectuées, cliquez sur "OK" pour appliquer les changements.
  2. Si vous avez apporté des modifications à l'onglet "Compte", assurez-vous de réinitialiser le mot de passe si nécessaire.

Étape 7 : Vérifier les Modifications

  1. Revenez à la liste des utilisateurs dans "Utilisateurs et ordinateurs Active Directory".
  2. Cliquez avec le bouton droit sur le compte utilisateur modifié et sélectionnez "Actualiser" pour vous assurer que les modifications sont prises en compte.

Félicitations ! Vous avez maintenant modifié avec succès les propriétés d'un compte utilisateur sur votre serveur Windows, y compris la configuration des onglets principaux, la définition des droits et des permissions nécessaires.

Tutoriel : Définition des Stratégies de Compte sur Windows Server

Étape 1 : Accéder à la Gestion des Stratégies de Sécurité

  1. Ouvrez le "Gestionnaire de serveur" sur votre serveur Windows.
  2. Naviguez vers "Outils" > "Stratégie de sécurité locale".

Étape 2 : Configurer les Paramètres de Verrouillage de Compte

  1. Dans le "Gestionnaire de stratégie de sécurité locale", développez l'arborescence à gauche.
  2. Accédez à "Stratégies de compte" > "Stratégie de verrouillage du compte".

Étape 3 : Paramètres de Verrouillage du Compte

  1. Faites un clic droit sur "Stratégie de verrouillage du compte" et sélectionnez "Propriétés".
  2. Dans la fenêtre des propriétés, configurez les paramètres de verrouillage du compte, tels que le nombre maximal de tentatives de connexion, la durée du verrouillage, etc.
  3. Cliquez sur "OK" pour appliquer les changements.

Étape 4 : Définir les Options d'Expiration de Mot de Passe

  1. Retournez à "Stratégies de compte" et accédez à "Stratégie de mot de passe".
  2. Faites un clic droit sur "Stratégie de mot de passe" et sélectionnez "Propriétés".

Étape 5 : Options d'Expiration de Mot de Passe

  1. Configurez les options d'expiration de mot de passe, telles que la durée maximale du mot de passe, la durée minimale du mot de passe, etc.
  2. Cliquez sur "OK" pour appliquer les changements.

Étape 6 : Vérification des Stratégies

  1. Pour vérifier que les stratégies ont été correctement configurées, utilisez l'outil de gestion des stratégies de groupe.
  2. Ouvrez la console "Stratégie de groupe" en tapant gpedit.msc dans la boîte de dialogue Exécuter.
  3. Naviguez vers "Configuration ordinateur" > "Stratégies" > "Paramètres Windows" > "Stratégies de compte" > "Stratégie de mot de passe" et "Stratégie de verrouillage du compte".

Étape 7 : Appliquer et Tester

  1. Les modifications apportées aux stratégies prendront effet après le prochain cycle de mise à jour des stratégies ou après un redémarrage du système.
  2. Testez les nouvelles configurations en tentant de vous connecter avec un compte utilisateur et en observant le comportement du verrouillage du compte et de l'expiration du mot de passe.

Félicitations ! Vous avez configuré avec succès les stratégies de compte sur votre serveur Windows, y compris les paramètres de verrouillage de compte et les options d'expiration de mot de passe.
