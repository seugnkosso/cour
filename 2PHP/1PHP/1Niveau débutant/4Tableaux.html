Tableaux en PHP

Tableaux Indexés

Les tableaux indexés utilisent des indices numériques pour accéder aux éléments. L'indice commence généralement à zéro.

php
<?php
// Création d'un tableau indexé
$fruits = array("Pomme", "Banane", "Orange");

// Accéder à un élément par son indice
echo $fruits[0]; // Affiche "Pomme"
?>

Tableaux Associatifs

Les tableaux associatifs utilisent des clés au lieu d'indices numériques pour accéder aux éléments.

php
<?php
// Création d'un tableau associatif
$personne = array("nom" => "Smith", "prénom" => "John", "âge" => 30);

// Accéder à un élément par sa clé
echo $personne["nom"]; // Affiche "Smith"
?>

Manipulation des Tableaux

Ajout d'éléments

php
<?php
// Ajout d'un élément à la fin d'un tableau indexé
$fruits[] = "Kiwi";

// Ajout d'un élément à un tableau associatif
$personne["genre"] = "Homme";
?>

Parcours des Tableaux

php
<?php
// Parcours d'un tableau indexé avec for
for ($i = 0; $i < count($fruits); $i++) {
    echo $fruits[$i] . "<br>";
}

// Parcours d'un tableau associatif avec foreach
foreach ($personne as $cle => $valeur) {
    echo "$cle : $valeur <br>";
}
?>

Suppression d'éléments

php
<?php
// Suppression d'un élément d'un tableau indexé
unset($fruits[1]);

// Suppression d'un élément d'un tableau associatif
unset($personne["âge"]);
?>

Fonctions de Tableau

PHP offre également des fonctions intégrées pour travailler avec les tableaux, telles que count(), array_push(), array_pop(), array_merge(), etc. Consultez la documentation PHP pour une liste complète.

Cela devrait vous donner une bonne introduction aux tableaux indexés et associatifs en PHP. N'hésitez pas à explorer davantage la documentation PHP pour des détails plus approfondis et des fonctionnalités avancées.

Fonctions de Tableau en PHP

1. count()

La fonction count() est utilisée pour compter le nombre d'éléments dans un tableau.

php
<?php
$fruits = array("Pomme", "Banane", "Orange");
$nombre_elements = count($fruits);
echo "Le tableau contient $nombre_elements éléments.";
?>

2. array_push() et array_pop()

array_push() ajoute un ou plusieurs éléments à la fin d'un tableau, tandis que array_pop() retire le dernier élément du tableau.

php
<?php
$fruits = array("Pomme", "Banane", "Orange");
array_push($fruits, "Kiwi", "Fraise");
$dernier_fruit = array_pop($fruits);
echo "Nouveau tableau : " . implode(", ", $fruits) . "<br>";
echo "Dernier fruit retiré : $dernier_fruit";
?>

3. array_shift() et array_unshift()

array_shift() retire le premier élément du tableau, tandis que array_unshift() ajoute un ou plusieurs éléments au début du tableau.

php
<?php
$fruits = array("Pomme", "Banane", "Orange");
$premier_fruit = array_shift($fruits);
array_unshift($fruits, "Kiwi", "Fraise");
echo "Nouveau tableau : " . implode(", ", $fruits) . "<br>";
echo "Premier fruit retiré : $premier_fruit";
?>

4. array_slice()

La fonction array_slice() extrait une portion du tableau.

php
<?php
$fruits = array("Pomme", "Banane", "Orange", "Kiwi", "Fraise");
$portion = array_slice($fruits, 1, 3);
echo "Portion du tableau : " . implode(", ", $portion);
?>

5. array_merge()

array_merge() fusionne deux tableaux ou plus en un seul.

php
<?php
$fruits1 = array("Pomme", "Banane");
$fruits2 = array("Orange", "Kiwi");
$nouveaux_fruits = array_merge($fruits1, $fruits2);
echo "Nouveaux fruits : " . implode(", ", $nouveaux_fruits);
?>

6. array_reverse()

array_reverse() renverse l'ordre des éléments dans un tableau.

php
<?php
$fruits = array("Pomme", "Banane", "Orange");
$fruits_inverses = array_reverse($fruits);
echo "Fruits inversés : " . implode(", ", $fruits_inverses);
?>

7. array_search()

La fonction array_search() recherche une valeur donnée dans un tableau et renvoie la clé correspondante si elle est trouvée.

php
<?php
$fruits = array("Pomme", "Banane", "Orange");
$position = array_search("Banane", $fruits);
echo "La Banane se trouve à la position : $position";
?>

8. array_keys() et array_values()

array_keys() renvoie toutes les clés d'un tableau, tandis que array_values() renvoie toutes les valeurs.

php
<?php
$personne = array("nom" => "Smith", "prénom" => "John", "âge" => 30);
$cles = array_keys($personne);
$valeurs = array_values($personne);
echo "Clés : " . implode(", ", $cles) . "<br>";
echo "Valeurs : " . implode(", ", $valeurs);
?>

Ces fonctions de tableau en PHP constituent une base solide pour la manipulation avancée des tableaux. N'hésitez pas à explorer davantage la documentation PHP pour découvrir d'autres fonctions et approfondir vos connaissances.
